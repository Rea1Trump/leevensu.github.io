新机器？

  当你入手一台新电脑的时候你要干啥？或者你觉得自己的电脑用着不舒服，或者系统出问题了，比如怎么都更新不了，当然重装是没有任何问题的，但是总归是要考虑兼容性，还有驱动安装的，还有系统激活，麻烦，所以，这里推荐用win10自带的恢复来解决，也就是Re：

Attention：重置前记得用U盘之类的备份重要数据，图片文件之类的，当然你的小电影，如果需要的话也可以考虑。

Tips：重置前可以考虑自行卸载一些软件，这样快得多，或者只重置C盘，完事后，再自行格式化其他盘，这样会更快，不要小看重置时间，有时候久的要命，机械硬盘的用户更要注意，不想等到花都谢了，采取一下建议，至于卸载软件，可以考虑geek uninstaller这个软件，下载就可以用，不用安装的，它的强制删除操作更快些，如果选择卸载的话，他会先触发该软件的卸载程序，中规中矩卸载，然后清理，慢一些。

  • 第一种：开机状态下按住Shift键然后点击重启就好了。
  • 第二种：点击【开始】->【设置】-> 【更新和安全】-> 【恢复】 -> 【开始】（重置此电脑下的开始）
  • 第三种：用win10的安装U盘（自己刻录=win10镜像（最好选择跟自己系统镜像一样的）+刻录工具+十多分钟），开机F12（不同电脑有不同）设置为U盘启动，然后进入了系统安装界面，左下角有一个修复计算机选项，点击。

PS：进入RE界面后有些不同，如果出现恢复那部分可以自行考虑，如果没有，那就找【疑难】这个选项，一定可以找到【重置此电脑】之类的东西，一般有两个选择，删除用户数据，把整个硬盘全部清空，时间较长，第二个就是保留用户数据，这样一般只清系统盘，更快些。

新机设置

  新买的电脑开机后先按照提示设置就好了，总没有义务教育的漏网之鱼吧，开机后第一步，不是下载你的女神，不是峡谷，更不是打开中国大学mooc，或者超星（我最讨厌你们这种为了学习不择手段的狠人(o゜▽゜)o☆）。

分区

  • 第一步：当然是先换一个壁纸欣赏一会儿，年轻人要冷静一点，不要着急。

  • 第二部：分区啊，总不能只有一个C盘。

    • 右键【此电脑】（桌面上没有的话，打开你的垃圾箱，侧边栏也有此电脑，都一样）->【管理】->【磁盘管理】：偏下的栏目有磁盘，哪里看着，只有一个C盘，右键一下然后->【压缩卷】,注意这个时候弹出来的是一个让人心慌的页面，注意几个数据：
      • 压缩前的总计大小（MB）
      • 可用压缩空间大小（MB）最多压缩多少，压多了，原来的盘就太小了
      • 输入压缩空间量（MB） 你需要修改的
      • 压缩后的总计大小（MB）
    • 你需要修改的就是（输入压缩空间量），记住单位是MB，我推荐你只要把原来的数字的第一位改一下就好了，开始改成原来的3/4，让C盘有整个盘的1/4.
    • 压缩完毕之后就会有黑色的未分配空间，选中未分配空间，右键新建简单卷，还是类似的分配空间给这个新的卷（自己有一个数就好，各占1/4，平均一下也未尝不可）过程中几乎没有要修改的，可修改一下卷标，就是一个标记而已。
    分区方案：
    • 512G=100+200+*G(其实没有512G大概只有477G左右，商家用1：1000这种计数方式，分三个区的话，大概这样，可以适当给C盘大一些，但别太大，浪费，最多150G，其他的基本这样)分区个数不需要太多，3-4个就好了，硬盘本身也不需要很大，512G很足够了，一般用户，1T的就根据自己的喜好分吧，
    • 混合硬盘分区的话，如果固态蛮大，那就分一下，分一个用来放常用的软件，或者大型软件和文档，这样处理比较快，机械部分用来当存储仓库一样，存不太常用的数据但是又不能缺少的，比如视频，或者某些软件的安装包，有时候网络环境不好，下软件很慢。

启动项修改

  打开【任务管理器】（点一下左下角的详细信息）->【启动】：右键选中一些启动项，禁用

推荐禁用：

  • OneDrive等。
  • 安装完毕软件之后还可以再来看看某些软件的神奇操作，就这么喜欢开机启动，为啥？
    • 当然不是因为喜欢你的电脑，他是馋你电脑的身子，开机启动就更能引起用户注意，弹个窗，来个通知啥的，来个更新啥的，然后稍微捆绑一点隐匿的操作。臭***。

文件管理

  • 在每个盘建好需要的文件夹，再安装软件的时候默认是C盘的路径，尽量进行修改，我是用D盘存软件，E盘装文件数据之类。
  • 建议在安装软件的时候先在D盘建好对应文件夹（尽量用英文名，或者拼音，尽量别用中文，有时候会很麻烦），建好文件夹有很多好处，最直观的是你知道装哪里，要手动删除的时候，也很方便，各种管理也很方便。

软件推荐

装机必备：

  • 浏览器：（上一篇已经推荐，此处从简）

    • Chrome：

      https://www.google.cn/intl/zh-CN/chrome/

      • 引擎：装好之后要自行修改默认搜索引擎，因为默认搜索引擎Google在中国是用不了的，修改为Bing。
      • 下载设置：修改为每次询问（这样更自由的下载文件，省的去移动），下载的默认文件夹可改可不改，默认在C盘，不改是因为文件资源管理器固定了下载目录的快速访问，这样很方便，（虽然也可以自己固定到快速访问，但是所有浏览器默认下载都一样，所有建议不改了）

    • Firefox：（注意搜索时写成：Firefox，别写成中文名：火狐，否则你只能看到国内版本cn）

      国际版：https://www.mozilla.org/en-US/firefox/new/

      国内版：http://www.firefox.com.cn/

      自己选择吧，国际版好像要修改默认语言，默认是英文的。需要在设置里修改

      • 引擎：如果你下载的是国内版（北京智谋公司的），默认引擎是百度吧，国际版应该是Google，国际版下载也不麻烦，就是很慢而已，看清楚链接是cn还是en-US，一样的找到设置，修改默认就好了。

      • 下载设置修改类似chrome。

      • 插件安装：火狐插件,

        插件官方链接：https://addons.mozilla.org/en-US/firefox/

        国内插件地址：https://addons.mozilla.org/zh-CN/firefox/

        国内插件地址更快些，但是没那么多插件，找不到插件时到官方链接就好了

    • Edge:（新版）下方提供下载链接，当然也可以用Bing国际版搜索Edge

      https://www.microsoft.com/zh-cn/edge

      • 其他修改类似

      • 插件：在Edge菜单栏的【拓展】，然后在左侧栏目有:

        从 Microsoft Store 获取扩展

  • 压缩与解压（win10自己可以解压某些压缩包，但不是很全，所以还是要自己专门下比较好）

  • 压缩解压软件的话，一般都是可以解压常见格式，但是压缩格式就不是都有，一般可以通过名字看出他最支持的格式，比如bandizip中是zip，winRAR是rar，7-Zip是7z至于选择的话，我用的是Bandizip。

        https://www.bandisoft.com/bandizip/

      • 卖相丑一点，但是7z压缩率很高，并且这是开源免费的

  • office：

    • 微软office下载：里面有【应用程序】，仔细找office2016，或2019都行，暂时没有365

    • Tim：QQ的另一个客户端，个人感觉比QQ要好，干净不臃肿，大力推荐，特别是团队，或者网课时期，这玩意对文件管理非常方便。
    • 微信：自己琢磨参考QQ

最后一讲：

暂时就这些，有喜欢的可以找我分享，但是还是再提一次：

  • 安装的时候记得自己建文件夹管理，尽量放别的文件夹，Chrome，Firefox等浏览器自动在C盘了就不要千方百计修改
  • 安装过程每一步都要注意，防止没看清小勾勾而出现捆绑问题，卸载的时候也要注意，防止百卸不成的尴尬
  • 最后安装好之后，一定记得设置，把自己能够修改的文件夹位置按照合理的方式迁移，把一些弹窗广告关了，还有自动升级，以及缓存清理等方面的策略要熟悉。以及一些适合自己的优化。
